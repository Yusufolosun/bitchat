## What is Bitchat?

Bitchat is a decentralized message board built on Stacks blockchain. Every message, pin, and reaction is stored permanently on-chain, providing true censorship resistance and user ownership.

## Features

### Core Functionality
- Post messages to global board
- Pin messages for extended visibility (24-72 hours)
- React to messages with on-chain reactions
- User statistics and leaderboards
- Transparent fee collection

### Security Features
- Emergency pause mechanism
- Spam prevention (6-block cooldown)
- Secure fee withdrawal
- Transferable ownership
- Complete event logging
- Time-based pin expiry
